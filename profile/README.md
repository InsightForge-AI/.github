## 👥 Team Structure

Our system is built using a **multi-team architecture**:

- 🧠 NLP Teams (A/B)
- 📊 ML Teams (A/B/C/D)
- 👁️ CV Teams (A/B)
- 🔎 RAG Team
- 🤖 Agent Team
- ⚙️ Backend Team
- 🎨 Frontend Team

Each team works independently but follows a shared **API contract**.

## 📡 Collaboration Rules

- Work only in your assigned module  
- Follow the API contract strictly  
- Use feature branches (`feature/<team>-<task>`)  
- Raise Pull Requests for all changes  
- No direct commits to `main`
